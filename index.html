1. Módulo Design: UX, UI e IxD.

Você vai mergulhar nos princípios essenciais de como criar interfaces digitais incríveis. Aprenda sobre a experiência do usuário (UX), a interface do usuário (UI) e o design de interação (IxD) de uma forma prática e envolvente.

2. Módulo Front-end: HTML, CSS e JS.

Domine as linguagens essenciais para criar páginas da web incríveis e interativas. Deixe sua marca na internet com habilidades em HTML para estrutura, CSS para estilo e JavaScript para funcionalidade.

3. Módulo ReactJS.

Pronto para dar vida às suas aplicações web? No Módulo ReactJS, mergulhe no poder do React para construir interfaces de usuário dinâmicas e responsivas. Explore componentes reutilizáveis e a abordagem declarativa que torna o desenvolvimento mais eficiente.

4. Módulo Back-end: NodeJS e MongoDB.

Construa aplicações do lado do servidor com NodeJS e explore a flexibilidade do MongoDB para armazenar dados. Desenvolva habilidades em programação assíncrona e bancos de dados NoSQL para criar aplicativos escaláveis e de alto desempenho.

5. Módulo Mobile: React Native.

Aventure-se na construção de aplicativos móveis nativos para iOS e Android, usando a versatilidade do React Native. Desenvolva habilidades para criar interfaces de usuário intuitivas e responsivas, aproveitando o poder de uma base de código compartilhada para agilizar o processo de desenvolvimento.
